INSTRUCTION TO RUN APPLICATION
1. Please Create DB with name "translatejawa_db"
2. Import database from folder "database/db_translatejawa.sql"
3. Please adjust database information in folder "inc/config.php"
3. Copy Source code PHP to htdocs
4. run with open browser and type : "http://localhost/translatejawa/"
5. You can use the apps.
